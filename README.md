🌱 Carbon Calculator for Households

A web-based application designed to help users estimate their household carbon footprint based on electricity consumption, transportation habits, and dietary choices. The project provides visual insights into carbon emissions and offers personalized recommendations to encourage sustainable living and environmentally responsible decision-making.

 --> Features
• Carbon footprint calculation
• Interactive and user-friendly interface
• Visual representation of emission data
• Personalized sustainability tips
• Responsive web design

🛠 Technologies Used
HTML, CSS, JavaScript, D3.js

🎯 Project Goal
To raise awareness about household carbon emissions and empower individuals to make eco-friendly choices that contribute to a greener future.

👨‍💻 Developed as part of the 1M1B Green Internship Program.
